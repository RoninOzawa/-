# 游戏平台运营 · 个人知识库

欢迎来到我的运营知识库。👋

这里是我从零开始搭建平台运营个人体系的完整记录。

我的第一份实习，是在一个用户体量小、资源有限的环境中参与各类运营。

虽然起初只是一段普通的实习经历，但碰上项目管理动荡，一个普本小实习生，却参与了许多决策项目。

当既然有了这千载难逢的机会，我自然积极争取，也因此在短短数月获得了大量的运营知识。

但运营职能范围何其之广，加之学习内容愈加繁杂，知识库散乱成点，虽然理论能说个大概，但决策起来不免虚浮。

希望能够通过整理知识体系，将碎片化的运营工作转化为可复用的SOP、数据工具和决策框架，形成个人的运营方法论。

建立这个仓库的初衷有三点：
1.  **自我梳理**：将实习期的思考与产出整理成可迭代的知识体系。
2.  **交流学习**：真诚地希望这些内容能帮助到同样在中小团队摸索的运营新人。
3.  **个人展示**：作为我求职的一份简历，直观地展示我的工作方法与成果。
